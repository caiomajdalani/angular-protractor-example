## Pre Conditions

1. Node.js LTS (8.9.4)

## Instalation

Install it as (in this sequence):

    $ npm install
    $ npm run updatewd

## Usage

Too run tests in terminal, type:

```
  npm test
  or
  protractor protractor.conf.js
```

To gerenate report (and clear the older one):

```
  npm run report
  or
  allure generate --clean
```

## Reports

1. To see reports:

```
  npm run displayReports
  or
  allure serve
```
